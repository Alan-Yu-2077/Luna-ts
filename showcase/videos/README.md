# 展示视频放这里

每个频道对应一个视频文件,放进本目录即自动播出(文件名必须一致):

| 频道   | 文件名       |
| ------ | ------------ |
| RAG    | `rag.mp4`    |
| Memory | `memory.mp4` |
| Tool   | `tool.mp4`   |
| Web    | `web.mp4`    |

- 推荐 16:9、H.264 编码的 `.mp4`(浏览器兼容性最好);其他比例会自动加黑边。
- 缺失的视频不会报错 —— 对应频道显示"节目准备中"测试卡。
- 想加新频道:在 `../main.js` 顶部的 `CHANNELS` 数组里加一项,再把视频放进来。
